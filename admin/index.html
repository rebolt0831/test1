Categories
How it works: use ▲▼ to set the order visitors see. Click Manage on a category to open it and add / edit / reorder its scenepacks. New scenepacks automatically bring their category to the top of the site for 7 days with a NEW badge. Nothing goes live until you hit Save all changes.
Thumbnails — three ways: hit Upload… to upload straight from your device (needs R2, see SETUP.md), put the file in the repo's images/ folder and type the path, or paste any https image URL. Changing a pack's Category dropdown moves it to that category. Remember to Save all changes.
